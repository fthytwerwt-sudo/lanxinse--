# 五结构成片主线锁定

状态：`已确认`
任务类型：`project_fact_lock_and_first_execution_plan`
事实层级：GitHub `main` 项目事实

## 1. 主线结论

`已确认`：澜心社自动剪辑当前首轮主线锁定为 `五结构连接成片版`。

本路线的目标不是先接入剪辑软件，也不是直接生成最终发布视频，而是先把可用素材整理成可复审字段，再从素材中形成五类结构片段候选，最后进入 1 条完整成片 Beta 的大纲与 EDL 草案。

本轮使用的来源优先级：

- `P0`：用户本轮明确输入，优先级最高。
- `P1`：当前仓库 `AGENTS.md`、机制包、既有项目事实与验证记录。
- `P2`：Google Drive 旧方案摘要与历史资料，只作为来源材料，不直接复制为当前事实。

旧 Drive 方案在本轮被整理为当前仓库可执行方案；其中历史边界“不进入 GitHub”已被本轮 `P0` 更新。

## 2. 五结构定义

| structure_type | 中文结构 | 子要素 | 当前用途 |
|---|---|---|---|
| `traffic_hook` | 引流型结构 | 痛点钩子 / 结果感 / 继续观看理由 | 建立观看理由，优先用于成片开头或强承接位置。 |
| `course_teaching` | 课程讲解型结构 | 问题 / 原因 / 方法 / 步骤 | 提供实质内容，承担主体讲解。 |
| `objection_response` | 异议回应型结构 | 用户质疑 / 解释 / 依据 / 引导 | 回应犹豫和质疑，不能夸大或替客户承诺效果。 |
| `case_trust` | 案例信任型结构 | 案例背景 / 变化过程 / 结果反馈 / 课程承接 | 建立信任，必须保留真实边界和风险复核。 |
| `conversion_push` | 促单转化型结构 | 权益 / 适合人群 / 风险解除 / 行动提醒 | 合规收束，引导行动但不制造虚假稀缺或效果保证。 |

无法判断或不适合强行映射的片段必须标记为 `unmapped_clip`。

## 3. 本轮不做项

`已确认`：本轮不连接 DaVinci，不做 DaVinci probe。

`已确认`：本轮不做美颜、字体、调色、多平台尺寸、批量模板化。

`已确认`：本轮不直接生成最终发布视频。

`已确认`：本轮不把一条样片写成批量稳定能力。

`已确认`：本轮不移动、不删除、不重命名原始素材。

`已确认`：本轮不把技术通过写成内容、审美、人感或业务通过。

## 4. 本轮目标

本轮只推进到以下执行入口：

1. 固化五结构连接成片版为当前主线。
2. 建立资料整理、素材配对、字段字典、阶段闸门和首轮执行清单。
3. 如果找到明确素材目录，只做只读文件级扫描，生成素材清单、配对表、缺失与异常、解析不明与字段缺口。
4. 为下一轮“首条成片 Beta 的片段解析任务”准备可复审输入。

本轮不进入真实剪辑、转码、渲染或批量自动化。
