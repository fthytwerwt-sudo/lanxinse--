# 阶段闸门与回退规则

状态：`已确认`
用途：防止资料整理、片段解析、成片判断和能力声明跑偏。

## 1. 总规则

技术可拼接不等于内容通过。

文件级扫描通过不等于素材可用。

五结构候选存在不等于成片成立。

单条 Beta 成立不等于批量稳定。

DaVinci 不影响当前五结构主线；本轮不连接 DaVinci，也不做 DaVinci probe。

## 2. 阶段闸门

| 阶段 | 进入条件 | 不通过时回退 |
|---|---|---|
| 素材验收 | 有明确素材目录、视频 / txt / 数据来源可追溯。 | 素材验收不过，不进入直播录屏解析。 |
| 直播录屏解析 | 有可用视频或带时间码转写，能建立 `source_id` / `segment_id`。 | 解析不过，不进入五结构映射。 |
| 五结构映射 | 至少能标出候选片段、风险和 `structure_type`。 | 五结构候选池不足，不进入成片连接。 |
| 风险复核 | 高风险业务事实、合规表达、客户确认项已列出。 | 风险复核不过，不进入样片微调。 |
| 样片微调 | EDL 草案、桥接字幕、人工终审记录可用。 | 样片微调不过，不进入轻量复跑。 |
| 轻量复跑 | 单条 Beta 问题可归因，并有第二样本或备用片段。 | 轻量复跑不过，不写批量稳定能力成立。 |

## 3. 必须回退的情况

- 素材缺完整直播录屏或可用转写。
- txt 无时间码，且无法可靠人工定位。
- 视频和 txt 文件名相似但无法确认对应关系。
- 五结构某类候选为空，且没有替代素材。
- 片段需要虚构补写才能连成闭环。
- 高风险表达未经过人工或客户确认。
- 用户 / 客户未做人审，却准备写审美、人感或业务通过。

## 4. 风险优先级

高风险优先于转化价值。

如果一个片段转化力强但存在以下风险，必须先进入人工复核：

- 价格、权益、退款或课程边界未确认。
- 效果承诺或夸大收益。
- 虚假稀缺、强诱导或敏感表达。
- 案例真实性和适用人群不清。
- 片段缺上下文，容易造成误解。

风险未消除前，不进入最终成片候选池。

## 5. 多线并行规则

多线并行可以压缩：

- 文件整理。
- 转写初筛。
- 时间线初分。
- 候选片段初选。
- 风险初标。

多线并行不能替代：

- 主控复核。
- 结构连接判断。
- 高风险业务事实确认。
- 用户人审。
- 客户业务确认。

## 6. 状态声明边界

允许写：

- `主线选择：已确认`
- `素材文件级整理：部分成立`
- `真实素材解析质量：待验证`
- `首条成片 Beta：待创建`
- `审美 / 人感通过：待用户人审`
- `业务事实通过：待客户确认`

禁止写：

- 自动剪辑稳定。
- DaVinci 已连接。
- 最终发布视频已生成。
- 审美通过。
- 业务事实通过。
- 单条样片证明批量能力成立。
