## 10. 可直接复用的仿片规则

1. 第一帧必须出现具体问题，不要只写“产后修复”。
2. 0-3 秒内完成“人群 + 痛点 + 一个动作 + 跟做入口”。
3. 3 秒左右必须进入动作，不能先讲长理论。
4. 教学主体保持稳定机位，动作清楚比转场更重要。
5. 字幕要跟动作口令同步，每句字幕只表达一个动作点。
6. 核心动作必须连续展示，不能靠口播跳过。
7. 换侧必须实拍，并用字幕明确“左右交替”。
8. 训练剂量必须高亮，例如“左右各1分钟”。
9. 专业词要配合身体感受，不要只堆“髋关节 / 盆底肌”。
10. 结尾用“做完感受一下”形成效果闭环。
11. 避免绝对疗效表达，把“告别困扰”改成更安全的“改善体态感受 / 缓解不适感”。
12. 后续仿剪不需要加箭头、贴纸或复杂转场；标题、字幕、动作清晰度是核心。
