## 6. 可复刻剪辑规则

1. 开头 0-1 秒必须出现病症/困扰关键词，不能慢热。
2. 0-3 秒内同时完成“人群 / 问题 / 结果承诺”：产后、骨盆外翻、腰胯宽、一个动作。
3. 第一个动作示范应在 3 秒左右出现，避免先讲太多理论。
4. 画面变化不靠特效，靠动作步骤推进：摆位、抱膝、晃动、换侧、回正。
5. 每 2-4 秒换一句底部字幕；字幕是主要节奏驱动。
6. 教学动作占比保持 80% 以上，主体必须能让观众跟做。
7. 承诺可以强在标题里，但结尾用“感受变化”比硬性疗效更稳。
8. 必须保留关键口令：脚后跟抬起、放在手臂上、双手抱膝、轻轻晃动、呼气时向内收、髋关节向内、盆底肌带动、左右各 1 分钟。
9. 字幕要大、清晰、带描边/阴影；关键剂量可用黄色高亮。
10. 机位要稳定，脚、膝、手臂关系必须清楚，不能只拍半身口播。
11. 换侧必须实际展示，不要只口头说“另一边一样”。
12. 结尾要给观众一个自我检查动作或感受问题，形成闭环。
