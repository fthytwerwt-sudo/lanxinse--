# Trae 分步执行 Prompt：直播切片 AI Skill 资料协助包

## 使用方式

请按顺序复制 Prompt 0 到 Prompt 7 到 Trae 执行。

执行原则：

- 只在本地执行。
- 不上传任何文件。
- 不删除任何原始文件。
- 不移动或改名原始文件，除非明确确认。
- 不知道就填“未知”或“待自动匹配”。
- 不要求人工分析专业剪辑结构。
- 只做资料整理、轻量标注、完整度检查。
- 自动匹配、结构拆解、AI Skill 稳定性均为待验证能力。

目标：

- 已剪成片文件夹。
- 如有发布数据、私信反馈、咨询反馈，可作为轻量参考。

- 成片编号。
- 成片文件名。
- 是否知道来源：知道 / 不知道 / 不确定。
- 可能来源直播：知道就写，不知道写“待自动匹配”。

## Prompt 5：负样本整理 prompt

```text
你是本地边界样本整理助手。请整理 30-60 条不希望 AI 学习的内容。

目标：
整理负样本。负样本是不希望 AI 学习或不希望 AI 自动切出来的内容，用来帮助 AI 学边界。

输入：
- 效果差成片。
- 私信少、咨询少、效果差的历史内容。
- 直播中不适合切的片段线索。

禁止：
- 不上传文件。
- 不删除、移动、改名原始视频。
- 不要求人工填写专业剪辑结构。
- 不把负样本理解为差评素材。

负样本可以来自：
- 私信少、咨询少、效果差的成片。
- 直播里不适合切的片段。
- 铺垫太长、跑题、重复、闲聊的内容。
- 有违规风险、画面差、声音差的内容。
- 明确不希望 AI 学习的内容。

请填写到 负样本表_negative_samples.csv：
- 负样本编号。
- 文件名或时间线索。
- 类型：效果差成片 / 不适合切片段 / 风险内容 / 其他。
- 私信 / 咨询反馈：少 / 无 / 未知。
- 为什么不要学：私信少 / 无转化 / 太水 / 重复 / 跑题 / 违规风险 / 画面声音差 / 不符合风格。
- 是否确定不要学：是 / 不确定。
- 备注，20 字以内，可不填。

输出：
- 30-60 条负样本记录。
- 哪些只是线索，后续需要确认。
- 哪些有敏感风险。

完成标准：
- 至少 30 条 / 组，最多建议 60 条 / 组。
- 正样本和负样本分开放。
- 不出现专业剪辑结构分析。
```
