# 直播切片 AI Skill 资料协助与云盘上传清单

资料协助版

请按本清单协助整理资料，并上传云盘供 AI 项目制作使用。

## 1. 本轮需要协助准备什么

| 模块 | 需要数量 | 需要协助准备什么 | 云盘中需要包含什么 |
|---|---:|---|---|
| 完整直播录屏 | 6 场 | 准备 6 场完整直播，覆盖效果好、普通、较乱三类 | 6 场原视频、直播清单、如有转写 / 字幕一起放入 |
| 正样本成片 | 60 条 | 准备希望 AI 学习的成片，优先选私信多、反馈好、认可度高的内容 | 60 条成片、正样本轻量标注表 |
| 负样本成片 / 片段 | 30-60 条 / 组 | 准备不希望 AI 学习的内容，优先选私信少、效果差、不适合切的内容 | 负样本表 |
| 对标视频 | 12-20 条 | 准备参考节奏、字幕、结构、包装的视频 | 对标视频、对标拆解表 |
| 盲测直播 | 2 场 | 单独预留，不参与前期规则整理 | 2 场盲测原视频 |
| 业务资料 | 1 套 | 准备课程、用户问题、核心优势、禁词、授权边界 | 已脱敏业务文档 |
| 脱敏与授权说明 | 1 份 | 确认哪些素材可分析、可留存 | 授权说明、脱敏检查表 |

必须记住：

- 这批资料是为了协助 AI 项目组制作直播切片 AI Skill。
- 全量素材不需要全部长期留存。
- AI 项目组需要保留指定样本，用于兜底、对照、纠错、验收、修 skill。
- 资料上传不等于 AI Skill 已稳定，稳定能力仍要后续验证。

## 2. 6 场完整直播怎么选

6 场必须覆盖：

| 类型 | 数量 | 选择标准 |
|---|---:|---|
| 效果较好 | 2 场 | 成交流程顺、互动好、已有多个优质切片 |
| 普通表现 | 2 场 | 正常直播状态，能代表日常水平 |
| 较乱或难度高 | 2 场 | 节奏乱、跑题多、评论复杂、声音或画面不稳定 |

每场直播最好包含：

- 完整画面。
- 声音。
- 评论区。
- 直播主题。
- 如有字幕 / 转写，也一起放入。

命名建议：

```text
LS001_日期_主题_效果好.mp4
LS002_日期_主题_效果好.mp4
LS003_日期_主题_普通.mp4
LS004_日期_主题_普通.mp4
LS005_日期_主题_较乱.mp4
LS006_日期_主题_较乱.mp4
```

另外单独预留 2 场盲测直播：

```text
BLIND001_日期_主题.mp4
BLIND002_日期_主题.mp4
```

这 2 场盲测直播不要参与前期规则整理，后面用于检查 AI Skill 是否能处理新直播。

## 3. 正样本成片怎么准备

正样本是“希望 AI 学习的成片”。

这里不要求配合方写剪辑专业分析。结构后续由 AI 自动解析生成，配合方只做轻量确认。

每条正样本只需要填写：

| 需要填写 | 怎么填 |
|---|---|
| 成片文件名 | 对应视频文件名 |
| 是否知道来源 | 知道 / 不知道 / 不确定 |
| 可能来自哪场直播 | 知道就写，不知道写“待自动匹配” |
| 大概线索 | 主题、主播话术、画面特征，可不填 |
| 私信 / 咨询反馈 | 多 / 中 / 少 / 未知 |
| 互动反馈 | 高 / 中 / 低 / 未知 |
| 发布效果 | 好 / 一般 / 差 / 未知 |
| 是否希望 AI 学这条 | 是 / 否 / 不确定 |
| 是否有敏感风险 | 有 / 无 / 不确定 |
| 备注 | 20 字以内，可不填 |

说明：

- 60 条正样本全部进入自动匹配池。
- 如果不知道来源，不用硬猜，统一写“待自动匹配”。
- 后续通过音频、字幕、画面相似度反查来源。
- 正样本越清楚，AI 越容易学到该保留的内容。

## 4. 负样本怎么准备

负样本是“不希望 AI 学习 / 不希望 AI 自动切出来的内容”。

它不是差评素材，而是帮助 AI 学边界。

负样本可以来自：

1. 已经发过但私信少、咨询少、效果差的成片。
2. 直播里不适合切的片段。
3. 铺垫太长、跑题、重复、闲聊、违规风险、画面 / 声音差的片段。
4. 对方团队明确觉得“不应该学”的内容。

每条负样本只需要填写：

| 需要填写 | 怎么填 |
|---|---|
| 文件名或时间线索 | 成片文件名，或直播里的大概时间 |
| 类型 | 效果差成片 / 不适合切片段 / 风险内容 / 其他 |
| 私信 / 咨询反馈 | 少 / 无 / 未知 |
| 为什么不要学 | 私信少 / 无转化 / 太水 / 重复 / 跑题 / 违规风险 / 画面声音差 / 不符合风格 |
| 是否确定不要学 | 是 / 不确定 |
| 备注 | 20 字以内，可不填 |

说明：

- 正样本和负样本必须分开放。
- AI 后续会对比“应该学什么”和“不应该学什么”。
- 目标是减少误切、乱切、切废话。

6. 是否允许 AI 项目组留存兜底样本确认。

## 6. 上传前检查

上传云盘前，按下表快速检查。

| 检查项 | 必须结果 |
|---|---|
| 6 场完整直播 | 已放入目录，覆盖效果好、普通、较乱三类 |
| 60 条正样本成片 | 已放入正样本目录，已填写轻量标注 |
| 30-60 条 / 组负样本 | 已放入负样本目录，已写明为什么不要学 |
| 12-20 条对标视频 | 已放入对标目录，已写明学什么 / 不学什么 |
| 2 场盲测直播 | 已单独放入盲测目录 |
| 字幕 / 转写 | 如已有，已放入字幕与转写目录 |
| 业务资料 | 已脱敏，不包含个人隐私 |
| 授权说明 | 已说明哪些样本可分析、可留存 |
| 文件清单 | 已填写并放入说明与清单目录 |

必须脱敏：

- 手机号、微信号、地址、订单号。
- 真实姓名、头像、评论区昵称。
- 私域二维码、支付二维码、后台截图。
- 未公开价格策略、内部资料、合作信息。
- 未授权人物、未授权品牌标识。

不确定是否能放入云盘的内容，先标记“待确认”，不要写成已确认。
